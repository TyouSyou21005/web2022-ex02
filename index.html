HTMLの「4 The elements of HTML」 から選んだ以下の要素を説明しようと思います。

  • 4.2.2 The title element
  • 4.2.4 The link element
  • 4.3.1 The body element

title要素とはそのページのタイトルを表す要素、ページのタイトルを指定する要素で、ページの「顔」とでも言うべきものです。 このtitle要素に記述した文字列は、ブラウザのタイトルバーやタブに表示されます。 コードは<title>”内容”</title>を書きます。 このページを例として、titleでは「日本大学文理学部情報科学科 Webプログラミングの演習課題02」ということです。

<head>
 <meta charset="utf-8">
 <title>日本大学文理学部情報科学科 Webプログラミングの演習課題02</title>
</head>

LINK 要素は開始タグのみで終了タグが存在しません。 このような要素の事を空要素(Empty Element)と呼びます。 空要素は「開始タグだけで終了タグが存在しない」とされ、 <link ...>と書くことになっていますが、その上位バージョンであるXHTML 1.0では、 空要素でも終了タグが必須となっており、<link ... />と記述しなければなりません。 コードでは<link rel="" hef="">です。

<link href="ex_web2.css" rel="stylesheet">

body要素とはそのページの内容（コンテンツ）を表す要素です。 画面に表示したいコンテンツは全てこの<body>要素内に配置します。 <body>要素は1つドキュメントに1つしか配置できません。 コードでは<body>"内容"</body>
